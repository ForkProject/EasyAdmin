EasyAdmin
=========

A base SmartAdmin(Bootstrap) template easy to use web admin system

## Development Environment
	a.jdk1.8  
	b.maven3  
	c.eclipse-jee-luna  
	d.tomcat7+  

## User Guide
  a.build  
	  cd /yourpath/EasyAdmin/  
	  mvn clean package -Pprod  
  b.deploy  
	 copy the easyadmin-web.war to tomcat webapps  
  c.init login user
